⚡ المميزات الأساسية (Core Features)

🧠

ذكاء اصطناعي محلي (Local AI)

يعمل كلياً على جهازك بموديل ذكاء اصطناعي محلي، بدون اشتراك ولا سحابة. كل البيانات سرية وآمنة. يدعم Ask Mode (للأسئلة) و Agent Mode (للتنفيذ)، تحليل الصور بالرؤية، وتحليل/استغلال الثغرات بزر واحد.

🔍

فحص 109+ نوع ثغرة

يكتشف أكثر من 100 نوع من الثغرات والمشاكل الأمنية — من XSS الكلاسيكي إلى Business Logic و Race Conditions — بمحرك تكيفي يقلل النتائج الخاطئة إلى الصفر تقريباً.

⚙️

مثل Burp Suite مجاناً

Intruder (Fuzzer متعدد الخيوط) · Repeater · Session Handler · Sequencer · WebSocket Scanner · Request Smuggling · Prototype Pollution.

🎯

اختراق ثنائي الهوية (IDOR)

كشف IDOR/BOLA بمقارنة استجابات حسابين (Victim + Attacker) لتأكيد الثغرة بشكل قاطع بدون شك.

🌐

OAST خارج النطاق

تكامل مع Interactsh لاكتشاف الثغرات العمياء (Blind SSRF · Blind CMDi · XXE OOB) عبر استدعاء خارجي.

🗺️

Site Map (خريطة الموقع)

شجرة URLs + جدول احترافي مثل Burp Target: يعرض كل المسارات والباراميترات والـ MIME والاستجابات وجميع الثغرات المكتشفة.

🤖 مساعد AI الشامل — الميزات المتقدمة

💬

وضع السؤال (Ask Mode)

يُجيب عن أي سؤال أمني مع إدراك كامل للسياق (نتائج المسح، الموقع، الجلسة). مدمج مع history محادثة (6 أدوار سابقة) + دعم الصور والملفات. اللغة: العربية فقط. يدعم Markdown والتنسيق.

🤖

وضع الوكيل (Agent Mode)

وكيل ذكي ينفّذ أوامر حقيقية — يكتب Python، يُرسل طلبات HTTP، يُشغّل اختبارات أمنية مخصصة، ويُحلل النتائج. 3 أدوات مدمجة: make_request لإرسال طلبات مخصصة، run_python لتنفيذ سكربتات، run_security_test لاختبار ثغرة محددة.

🖼️

تحليل الصور بالرؤية الاصطناعية (Vision)

رفع صورة شاشة (Png/Screenshot) → النموذج البصري يستخرج منها تلقائياً: الـ URL ومصدر الثغرة، نوع الثغرة (VULN)، الـ Payload الدقيق الظاهر. ثم الـ Agent يُعيد إنتاج الـ Payload ويختبره على الهدف الحقيقي.

🔬

تحليل الثغرة بالذكاء الاصطناعي ⚡

زر 🤖 Analyze with AI في كل Finding — يُرسل الثغرة كاملةً (URL، Payload، Evidence، Detail) لمحرك AI_analyzer.py ليُصنّفها: True Positive vs False Positive مع Confidence % ونسبة الخطورة واستغلال محتمل. يقلّل الـ False Positives إلى أقصى درجة.

⚡

استغلال الثغرة تلقائياً ⚡

زر ⚡ Exploit & Test في كل Finding — يُشغّل محرك exploit_engine.py ليختبر الثغرة فعلياً ضد الهدف الحقيقي (Real-time Exploit Verification). يتأكد إذا كانت الثغرة قابلة للاستغلال أم لا ويعيد PoC عملي.

🔄

Agent Auto-Fix & Payload Generator

AI يُولّد Exploit POC تلقائياً لكل ثغرة (generate_exploit). الـ Agent يُصحّح ويُعدّل الـ Payload بناءً على استجابة السيرفر، يعيد المحاولة حتى النجاح، ويحفظ كل شيء في مجلد الهدف.

🏆 القوة الحقيقية للبرنامج

  • 🧠 ذكاء اصطناعي محلي بالكامل AI موديل ذكاء اصطناعي متقدم يعمل محلياً بالكامل — لا اشتراك، لا سحابة، ولا تسريب للبيانات. يدعم Mode مزدوج: Ask (للأسئلة) و Agent (للتنفيذ المباشر للأوامر).
  • ⚡ Agent ينفذ أوامر حقيقية داخل التيرمنال AI يكتب ويشغل كود Python مباشرة، يرسل طلبات HTTP حقيقية، يبني اختبارات أمنية مخصصة، يحلل النتائج، ويكتب تقارير — كل ذلك داخل التيرمنال دون مغادرة البرنامج. 3 أدوات مدمجة: make_request, run_python, run_security_test.
  • 🔬 Analyze with AI — تصنيف الثغرة ذكياً AI كل Finding لديه زر 🤖 Analyze with AI يُرسل التفاصيل (الـ URL والـ Payload والإيفيدنس) لـ ai_analyzer.py. الـ AI يُصنّفها True/False Positive مع Confidence%، ويُحدد الـ Exploit Impact وخطورة الاستغلال بدقة عالية — يقلّل النتائج الخاطئة بشكل جذري.
  • ⚡ Exploit & Test — تأكيد الاستغلال عملياً Exploit زر ⚡ Exploit & Test يُشغّل محرك exploit_engine.py لاختبار الثغرة ضد الهدف الحي مباشرةً. يتأكد إذا كانت الثغرة قابلة للاستغلال (exploitable) أم لا، يُعيد PoC عملي + إثبات الاستغلال + يولد Exploit POC تلقائياً لكل ثغرة.
  • 🎯 تشغيل متوازي — بدون تجميد Performance الـ Scan و Chat يعملان معاً في آن واحد. 24 Thread Pool للفحص + 1 Thread مخصص للمتصفح + فصل كامل للـ Executor. البرنامج لا يتجمد أو يعلق أبداً أثناء الفحص، ويمكن تصفح النتائج والتحدث مع AI في نفس الوقت.
  • 📡 بث مباشر لحظي (SSE) Real-time نتائج المسح تظهر فوراً في 4 أماكن مختلفة في وقت واحد: Terminal · Live Dashboard · Site Map · Vulns Panel — بتقنية Server-Sent Events مع تحديث فوري بدون تأخير.
  • 🕷️ زاحف ذكي بمتصفح حقيقي (Crawler) Engine يستخدم متصفح Chromium حقيقي عبر Puppeteer لتصفح المواقع وتنفيذ JavaScript بالكامل. يكتشف كل الروابط، النماذج، API endpoints، وحتى الـ SPA (Single Page Applications) التي لا تظهر للمتاحف العادية.
  • 🔒 كشف الأسرار والمفاتيح الخاصة Secrets يكشف تلقائياً مفاتيح AWS (AKIA…), Google API, GitHub Tokens, Stripe Live Keys, JWTs, Private Keys (RSA/EC/OPENSSH), Slack Tokens, Twilio, SendGrid في ملفات JS والاستجابات — مع تحليل الأنتروبي (Shannon) لتصفية النتائج الخاطئة.
  • 📊 كشف دقيق بدون false positives Accuracy يُرسل طلباً أساسياً (Baseline) أولاً لقياس الوضع الطبيعي، ثم يقارن الفرق بدقة: XSS يتحقق من الانعكاس الحرفي غير المشفر، SSTI يتأكد أن التعبير حوسب (الرقم ظهر والنص الأصلي اختفى)، SQLi(time-based) يقارن التوقيت مع الأساس (threshold +4s).
  • ⏯️ Pause / Resume — إيقاف واستئناف ذكي UX يمكن إيقاف المسح في أي وقت واستئنافه تلقائياً. حتى لو أغلقت المتصفح بالكامل أو حدث تحديث، البرنامج يستعيد الحالة كاملة من localStorage ويستأنف المسح من حيث توقف دون فقدان أي تقدم.
  • 📝 تقارير احترافية جاهزة للتسليم Reports يُصدر تقارير HTML + JSON كاملة مع الـ Severity (Critical/High/Medium/Low/Info) والتفاصيل والأدلة (Evidence) والتوصيات (Recommendations) — جاهزة للتسليم للعميل أو إرفاقها مع تقرير Bug Bounty.
  • 🎯 HackerOne Scope Fetcher Recon يسحب نطاق برنامج HackerOne تلقائياً (In-scope + Out-of-scope) عبر 3 طرق: __NEXT_DATA__ JSON، data-asset-identifier، والتعبيرات المنتظمة. يحدد أنواع الثغرات المسموحة والـ Bounty المتاحة.
  • 🖼️ AI Vision — تحليل الصور واللقطات Vision يدعم موديلات الرؤية (LLaVA, Qwen2-VL) لتحليل صور الشاشات واستخراج الـ URLs والـ Payloads ونوع الثغرة تلقائياً من الصورة — بدون كتابة يدوية.
  • 🗂️ Project Management + File Browser Tools إدارة المشاريع والملفات من داخل البرنامج. Browse لأي مجلد على الكمبيوتر، عرض شجرة الملفات، فتح وتحرير، وتنفيذ AI Task على أي ملف مباشرة.
